本文是利用深度学习做时间序列预测的入门教程，用到的模型包括卷积神经网络（CNN）和循环神经网络（RNN）。全文分为两大部分，又可以细分为：

  • 预测单个时间步：
    • 预测一个特征。
    • 预测所有特征。
  • 预测多个时间步：
    • 单发预测：模型跑一次输出所有时间步的结果。
